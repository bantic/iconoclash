In the process, Iconoclash created custom CSS properties for all SVG elements that had an ID attribute specifying CSS properties to expose. For example, consider this before and after:


Input: <g fill="#E2574C" id="iconoclash fill">
Output:  <g fill="#E2574C" style="fill: var(--vespa-g1-fill)">

Here's how you can use them in your page!

{{#icons}}

{{id}}

HTML embed:

You can add the {{id}} icon into an HTML file like this.

<svg width="100" height="100"><use href="icons.svg#{{id}}"></use></svg>

Configurable CSS Properties

These CSS properties were exposed from the SVG for configuration. Try editing them!

CSS background embed:

If you only want the icon as a background image, add a class of iconoclash-{{id}} to any element:

<div class="iconoclash-{{id}}"></div>
