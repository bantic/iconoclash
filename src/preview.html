You can add the {{id}} icon into an HTML file like this.

<svg><use href="icons.svg#{{id}}"></use></svg>

You can also add the icon as a background image by adding a class of iconoclash-{{id}} to any element. CSS customizations won't apply to backgrounds, however.

<div class="iconoclash-{{id}}"></div>

Sample animation!

the conveyor has an animation property exposed on one of its groups. It's set to initial by default, but if you set it to a keyframe animation, it'll animate.


<svg class="animateddemo">
  <use href="icons.svg#conveyor"></use>
</svg>

<style>
@keyframes conveyor {
  0% { transform: translateX(0px); }
  100% { transform: translateX(-30px);}
}
.animateddemo {
  --conveyor-g4-animation: conveyor 3s infinite;;
}
</style>
